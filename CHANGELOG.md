# Changelog

All notable changes to the "Markdown Ninja" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-23

### Added
- Initial release
- Notion-like WYSIWYG editing for Markdown files
- VS Code theme integration (light/dark)
- Auto-save functionality
- Slash commands menu (`/`)
- Keyboard shortcuts (Cmd+B, Cmd+I, Cmd+U)
- Callout blocks (info, warning, error, success)
- Code blocks with language selection
- Interactive checkboxes
- Tables support
- Images support
- Bullet and numbered lists
- Headings (H1, H2, H3)

### Technical
- Built with BlockNote editor
- Uses VS Code CSS variables for theming
- Debounced auto-save (300ms)
